## 调用方法：

```
/* 需要初始化一下容器 */
   $.initImg();
/* 调用 */
    $('.imgEnlarge').imgEnlarge();
    $('.imgEnlarge2').imgEnlarge();
    /* 修改出现和隐藏的时间 */
    $('.imgEnlarge3').imgEnlarge({interval:2000});
```
